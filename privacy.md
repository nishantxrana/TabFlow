TabFlow Privacy Policy

TabFlow is a local-first browser extension. All tab session data is stored locally on the user’s device using Chrome’s storage APIs.

Data Collection:
TabFlow accesses open tab URLs and titles solely to allow users to save and restore tab sessions. This data is not transmitted to any external servers.

Data Sharing:
TabFlow does not sell, transfer, or share user data with third parties.

Cloud Features:
Cloud sync and optional features may be introduced in the future. If implemented, this policy will be updated accordingly before release.
